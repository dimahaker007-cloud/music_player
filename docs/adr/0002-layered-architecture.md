Структура шарів та їх відповідальності
-
Api layer<br>
Відповідальність:
+ Обробка HTTP запитів та відповідей
+ Аутентифікація та авторизація 
+ Обробка помилок та трансформація у відповідні HTTP статуси

Конкретні компоненти:
+ musicController - REST endpoints для управління піснями
+ userController - REST endpoints для управління користувачами

Bussines layer<br>
Відповідальність:
+ Створення запитів до бази даних через Services
+ Обробка помилок звернень до бази даних

Конкретні компоненти:
+ UserService
+ MusicService
+ MusicPlayerService

Core layer<br>
Відповідальність:
+ Зберігані всіх методів та DTO
+ інтерфейси для сервісів

Конкретні компоненти:
+ IUsers
+ IMusic
+ DTO
+ models

frontend layer<br>
Відповідальність:
+ реалізація робочого інтерфейсу
+ сворення запитів до контролерів

Конкретні компоненти:
+ blazor

Напрямок заллежностей: frontend > Api > Bussines > Core

Заборонені звязки: Core > Bussines > Api
